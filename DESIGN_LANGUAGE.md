# Design Language

## Direction

Calm, geometric, premium, and commercially believable.

## Principles

- High trust over decorative noise
- Strong hierarchy and spacing rhythm
- Light surfaces with soft contrast
- Minimal borders and restrained motion
- Mobile-first structure

## UI rules

- Avoid generic dashboard clutter
- Keep forms short and contextual
- Use status pills for operational truth
- Prefer one primary action per panel
- Keep payment and admin surfaces distinct

## Visual system

- Warm neutral background
- White glass-like surfaces
- Deep neutral text
- Black primary action
- Subtle shadows, no heavy chrome
